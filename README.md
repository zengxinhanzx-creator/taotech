## 快速開始

1. **下載文件**：將所有文件保存到同一目錄
2. **打開網站**：雙擊 `index.html` 文件在瀏覽器中打開
3. **本地服務器**（推薦）：
   ```bash
   # 使用Python啟動本地服務器
   python -m http.server 8000
   
   # 或使用Node.js
   npx serve .
   ```
